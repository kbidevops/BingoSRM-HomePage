BingoCVM — 취약점진단 자동화 플랫폼

정보통신기반시설 및 클라우드 시스템 취약점진단 자동화를 제공하는 KBi의 대표 보안 솔루션입니다. 진단 / 평가 / 분류 / 조치 / 이력 / 보고서까지 원스톱으로 제공합니다.

01
국내 컴플라이언스 규정 준수
KISA 취약점점검 가이드 기준에 따른 체계적인 점검으로 법적 요건을 충족합니다.
02
ISMS-P · CSAP 인증 준수 관리
정보보호관리체계 인증과 클라우드 보안 인증 준수를 위한 체계적 지원을 제공합니다.
03
공공기관 사이버보안 실태평가 대응
정부·공공기관의 사이버보안 실태평가 요구사항에 맞춤 대응합니다.
04
독보적 ATC 자동화 엔진
자산 등록부터 진단·수집·분석·조치까지 전 과정을 자동화하여 담당자 업무시간을 대폭 단축합니다.
05
On-Premise + Cloud 통합 진단
온프레미스와 클라우드 자산을 하나의 플랫폼에서 CCE + CVE 취약점 진단이 가능합니다.
